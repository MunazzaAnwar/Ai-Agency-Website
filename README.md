# 🤖 AI & Software Development Agency Website

A premium, multi-page agency website built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

🌐 **Live Demo:** [munazzaanwar.github.io/Ai-Agency-Website](https://munazzaanwar.github.io/Ai-Agency-Website/)

---

## 📌 Overview

This project is a fully responsive, production-ready agency website designed for an AI & Software Development company. It features a modern dark theme with glassmorphism UI, smooth scroll-triggered animations, and interactive elements — inspired by award-winning designs on Awwwards.

---

## 🛠 Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Semantic structure, SEO-friendly markup |
| CSS3 | Glassmorphism, animations, responsive layout |
| JavaScript (Vanilla) | Interactivity, DOM manipulation, animations |

---

## ✨ Features

### UI & Design
- 🌑 Modern dark theme with glassmorphism effects
- 💡 Mouse-following glow / spotlight effect
- 🔤 Typing text effect in the hero section
- 🎨 Premium typography and consistent spacing

### Pages
- **Home** — Hero, stats, highlights
- **About** — Company story and mission
- **Services** — AI, Web, Mobile, Cloud, UI/UX
- **Projects** — Filterable project showcase
- **Blog** — Article cards layout
- **Contact** — Validated contact form

### Animations & Interactions
- Animated hero section with floating elements
- Scroll-triggered fade-in and slide-up animations
- Parallax scrolling effects
- Animated statistics counters
- Service cards with hover transformations
- Client testimonials slider
- FAQ accordion

### Other
- Sticky navigation bar
- Attractive loading / splash screen
- Newsletter subscription section
- Footer with social links
- Fully responsive (mobile, tablet, desktop)
- SEO-friendly HTML structure

---

## 📁 Project Structure

```
Ai-Agency-Website/
├── index.html       # Main HTML file (all pages as sections)
├── style.css        # All styles, animations, responsive rules
└── script.js        # Interactivity, counters, slider, animations
```

## 🚀 Getting Started

No build tools or package manager needed.

```bash
# Clone the repository
git clone https://github.com/munazzaanwar/Ai-Agency-Website.git

# Open in browser
open index.html
```

Or simply open `index.html` directly in any modern browser.
